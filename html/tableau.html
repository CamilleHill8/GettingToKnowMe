What is Tableau?

Tableau is a premuim data analysis software system that enables individuals to visualize data while also making interactive graphs for users and viewers.

Check out their website here!

Tableau also has a feature called Tableau Public that allows you to upload live data visualizations that can then be hosted on websites. Take a look below at some of the visualizations I created on Tableau Desktop and then linked on this website!
